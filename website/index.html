Welcome to Cabot.


Cabot is a simple, but powerful, modular Discord bot written in Discord.js.


It's purpose is to be a useful and fully free bot for servers. Currently, it is still in development.

We currently have these features:

    Fun

  • 8 Ball
  • Colors
  • Repeat After Me

    • Moderation

  • Kicking, Banning, and Warning a user
  • Per server logging and config
  • And a few easter eggs...

20%

Cabot's development is 20% done. More features will come as development continues.
